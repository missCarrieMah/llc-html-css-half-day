##HTML / XHTML / HTML5 What's the difference? **HTML (HyperText Markup Language)**: Used to structure and define page content. **XHTML (Extensible HyperText Markup Language)**: Reformed HTML based on XML. Syntax rules are more strict. **HTML5 (HyperText Markup Language, revision 5)**: Most current version. Improved the support for multimedia and introduced new structural tags. Not supported in older browsers. Basically, these three terms refer to different versions of the same language but are often just called HTML interchangeably. The [World Wide Web Consortium](http://www.w3.org/TR/html5/) (W3C) governs these web standards with specifications, guidelines, tools and recommendations.

##Good File Naming Habits * Use lowercase letters. (e.g. about.html instead of About.html or ABOUT.html) * Avoid s p a c e s. Underscores (_) or dashes (-) are valid. However, dashes are generally preferred for SEO (search engine optimization). (e.g. business-hours.html instead of business hours.html - The browser will interpret it as business%20hours.html) * Avoid symbols like # & * in your file name. * Keep your file names concise, yet meaningful. (e.g. about.html, contact.html instead of page1.html, page2.html) These file naming tips can be applied to any files such as images, folders, CSS, etc.
